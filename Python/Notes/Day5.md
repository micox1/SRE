# Functions and Returns 

## Intro to Functions
- Functions are a convenient way to group our code into resuable blocks. A function contains a sequence of steps that can be performed repeatedly throughout a program without having to repeat the process of writing the same code again. 

- A function consists of many parts 
  - The `def` keyword indicates the beginning of a function. The fucntion header is followed by a name in the `function_name` format that describes the task the function performs. It is best practice to give your function a descriptive yet concise name
  - Following the function name is a pair of parantheses `()` tha can hold input values know as **parameters**
  - A colon marks the end of a function header 
  - Lastly we have one or more valid Python statement that make up the function of the body 

```python 
def function_name():
```

- To call a function we must type out the function's name follwoed by a pair of parantheses and no indentations

## Function Parameters 
- Function parameters allow our function to accept data as an input value. We list the parameters a function takes as input between the parantheses of a function ().

```python 
def my_function(single_parameter):
```
